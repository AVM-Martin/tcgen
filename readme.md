# Tcgen
## TcFrame Converter to Kattis Problem Format



### Installation

  * Follow the tcframe installation manual and add the following command into ```~/.bashrc```

```bash
export TCGEN_HOME=path/to/tcgen/folder
alias tcgen=$TCGEN_HOME/scripts/tcgen
```
